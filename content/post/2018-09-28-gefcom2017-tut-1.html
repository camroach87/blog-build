Update (28 January, 2020). I have decided to post the paper’s code in dedicated GitHub repositories instead of a tidied up walkthrough.

Code, R markdown files and a preprint of the paper Reconciled boosted models for GEFCom2017 hierarchical probabilistic load forecasting (Roach 2018) can be found at my 1701-gefcom github repository.

A dedicated package that includes the tidied competition data in .rda format can be found at my gefcom2017data repository. This repository also includes the original excel spreadsheets released by ISO New England, US federal holidays and the functions used to tidy and format the raw data.

References

Roach, Cameron. 2018. “Reconciled Boosted Models for GEFCom2017 Hierarchical Probabilistic Load Forecasting.” International Journal of Forecasting. doi:10.1016/j.ijforecast.2018.09.009.
